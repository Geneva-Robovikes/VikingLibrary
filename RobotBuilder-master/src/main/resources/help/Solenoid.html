Solenoid (PCM)

What is it?

A solenoid that is plugged into the Pneumatics Control Module (PCM). It controls a pneumatic cylinder to create linear motion.

Properties

Output PCM (Solenoid)
The PCM CAN ID that this solenoid is plugged into.
Output Port (Solenoid)
The port on the PCM that this solenoid is plugged into.
