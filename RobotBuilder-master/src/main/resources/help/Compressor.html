Compressor

What is it?

The compressor keeps the pneumatics of the system pressurized up until the digital switch detects that there is too much pressure.

The PCM will run the compressor automatically as long as there is a solenoid constructed. The compressor class is only required to turn off the compressor when it would normally be running

Properties

PCM ID
The Pneumatics Control Module (PCM) CAN ID that the compressor is plugged into.
