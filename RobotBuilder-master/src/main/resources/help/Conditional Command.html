Conditional Command

What is it?

A conditional command runs one of two commands. The command to run is chosen at run time by the condition() method.

Properties

OnTrue Command
Command to run when condition() returns true
OnFalse Command
Command to run when condition() returns false
Button on SmartDashboard
Whether or not to automatically place a button on the SmartDashboard.
Parameters
Variables to be passed into the command. Data can be of types boolean, byte, int, long, double, or string.
Run When Disabled
Sets whether or not this Command should run when the robot is disabled.
